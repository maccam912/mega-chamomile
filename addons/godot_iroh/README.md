# Vendored godot-iroh runtime

This directory contains the desktop binaries from
[`tipragot/godot-iroh` v0.1.5](https://github.com/tipragot/godot-iroh/releases/tag/v0.1.5),
upstream commit `1f15dd4d82913aa9cf3af3a2738ec7006f4ddc70`.

- Upstream archive SHA-256:
  `a6166d2e7ade8057b9bfc90e46a1f70dac0c33950a31ae7e1843be5289ce349d`
- Included targets: Linux x86_64, Windows x86_64, and universal macOS.
- License: MIT; see `LICENSE` in this directory.

The binary files are stored with Git LFS. ENet remains an independent option;
the game checks for the native iroh classes before offering code-based rooms.
